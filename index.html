Export

Use can export the annotated results as a CSV file using the Export button below. The button becomes active once all computations are complete.



File Details

Chrome will download the file as expected and save it automatically. Firefox and Safari will save the file, with an incorrect extension. Rename the file data.csv and it will open. Internet Explorer will pop open a new window and you will have to use 'Save As' to save the generated content. Save the file with a .csv extension. The CSV file will open in Excel, Numbers, Open Office, Google Docs, etc.

The exported CSV file adds 8 columns to the downloaded Twitter data: polarity.value, polarity.confidence, emotion.value, emotion.confidence, intensity.value, intensity.confidence, spam.value, spam.confidence. These columns are the ConveyAPI computed values for each Tweet.
